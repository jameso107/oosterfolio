# James Oosterhouse Portfolio

A modern, responsive portfolio website showcasing my work as a Robotics Engineer and Leader, built with a maize and blue color scheme inspired by the University of Michigan.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Sections**:
  - Hero section with introduction
  - About section highlighting personal and professional identity
  - Featured projects (T-shirt Cannon Robot, Ballbot, Haunted House)
  - Journey timeline
  - Contact information

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required - it's ready to go!

## Customization

### Colors
The color scheme uses University of Michigan colors defined in CSS variables:
- Maize: `#FFCB05`
- Blue: `#00274C`
- Blue Light: `#003A6B`
- Blue Dark: `#001A33`

You can modify these in the `:root` section of `styles.css`.

### Content
Edit the HTML file to update:
- Personal information
- Project descriptions
- Contact links
- Journey timeline items
